Hoos Got the Time?

Do you?

Field Information

  • Class Time Per Credit
    • Hours of class time per credit taken.
    • UVa policy is >= 50 minutes of class per credit taken.
  • Homework Time Per Credit
    • Hours of time spent on non-in-class work per credit taken.
  • Internship Time
    • Hours per week spent at an intership or internship-like job.
  • Other Commitments
    • Hours per week used for other commitments (clubs, athletics, etc...)
  • Credits Taken
    • The number of credits.
  • Average Start/Stop
    • Average time tasks start and stop being accomplished.
    • Classes count as tasks.
  • Static vs. Dynamic Load
    • Static Load is time devoted to tasks that occur at a fixed time (i.e Class)
    • Dynamic Load is time devoted to tasks that can be compleated at any time (i.e Readings, homework.)
  • Active vs. Dynamic vs. Free Time
    • Active Time is time the student is *able* to complete tasks (i.e awake).
    • Dynamic Time is time active the student has where any non-static task can be accomplished.
    • Free Time is active time the student need not spend compleating any type of task.
